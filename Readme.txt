Hello!

This is the tool to mask the PII in data. The Tool will take speech, audio and text as input, and generates result file with adding corrected text and annotated text columns as results.

Input Requirements:

	*Audio file must be in the ".wav" format
	*Text file must be in the ".csv" format with column name as "text"  


Requirements:

Gingerit
spacy
csv
pandas
nltk
speech_recognition


How to Run?

From windows command prompt:
python translator.py <file name>
